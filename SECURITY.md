# Security Policy

## Supported versions

Dataset Finder is under active development. Security updates are applied to
the latest release and the current `main` branch.

| Version | Supported |
|---|---|
| 0.2.x | Yes |
| 0.1.x | No |

## Reporting a vulnerability

Please do not publicly disclose a suspected vulnerability before the
maintainer has had a reasonable opportunity to investigate it.

Send a private report to the repository maintainer through the contact
information available on the maintainer's GitHub profile.

Include:

- a description of the vulnerability;
- affected files or functions;
- steps required to reproduce it;
- the potential impact;
- any suggested remediation.

Do not include real credentials, access tokens, protected health
information, or confidential research data in a report.

## Scope

Relevant reports include:

- unsafe handling of credentials or tokens;
- command or code injection;
- insecure temporary-file handling;
- dependency vulnerabilities with a demonstrated effect on this project;
- unintended disclosure of private data.

General feature requests and ordinary software defects should be submitted
through GitHub Issues.
